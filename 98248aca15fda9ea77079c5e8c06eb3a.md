# Go zero values
Variables declared without an explicit initial value are given
the _zero value_.

The zero value is:
- 0 for numeric types;
- `false` for the boolean type;
- (the empty string) for string.

E.g.
```go
package main
import "fmt"
func main() {
var i int
var f float64
var b bool
var s string
fmt.Printf("%v %v %v %q\n", i, f, b, s)
}
```
